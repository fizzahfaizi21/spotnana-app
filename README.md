# Spotnana Frontend Project

## Overview

This project was built as part of a Frontend Engineer technical
assessment designed to simulate a real-world scenario. The goal was
to create an AI-integrated web application where users can input
prompts, send them to an AI API, and dynamically receive and display
responses. 

Using Next.js with the App Router, I focused on building a clean,
intuitive UI within a structured architecture that demonstrates
effective problem-solving, API integration, and clear communication
between frontend and backend layers.

To match Spotnana(a travel company), this app lets its users input
in any city as a destination, upto three interests, and any other
extra notes. The AI then builds you an itinerary for the entire day,
naming all the business and locations it recommends. 

## Tech Stack

* Next.js
* TypeScript
* Node.js (via Next.js API routes)
* CSS Modules

## API

* AI planning/summary = OpenAI (gpt-4o-mini)
